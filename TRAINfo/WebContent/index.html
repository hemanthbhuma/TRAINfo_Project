About Us



Revanth Vermareddy

This is my first complete bootstrap Project in Advanced Java Learing.


This is an Online Portal to view the status of train by using the Train Number. If we don't know the Train Number travelling from a particular Station to another station there is no problem, you can check the available trains based on the starting and ending point of the journey. If the trains are not available then it displays No Trains available notification.


User can view his details which he entered during his registration in the portal and if their are any modifications to be done he can do so through edit profile option. The user can then finally logout throgh his portal.
